# Raydrift Meta

SEO-first Astro components with Schema.org JSON-LD helpers.

## Structure

- `apps/web`: Astro site
- `packages/primitives`: Published package and CLI

## Quick start

```bash
pnpm dlx raydrift-meta init
```

Creates `src/seo.config.ts` in your app. See the package readme for examples and API docs.
